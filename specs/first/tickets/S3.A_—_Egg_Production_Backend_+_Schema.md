## DB Schema

New file file:lib/db/src/schema/egg-production.ts — export from file:lib/db/src/schema/index.ts.

Three tables per file:specs/05_EGG_PRODUCTION.md §3:

| Table | Key columns |
| --- | --- |
| `egg_collections` | `batch_id`, `collection_date`, 6 grade counts (s/m/l/xl/cracked/dirty), `total_count` (generated), `good_count` (generated), `population_at_collection`, `production_rate_bp` |
| `egg_discards` | `batch_id`, `discard_date`, `count`, `reason` enum, `withdrawal_record_id` |
| `egg_sales` | `batch_id`, `sale_date`, `crates_sold`, `looses_sold`, `price_per_crate_pesewas`, `price_per_loose_pesewas`, `total_revenue_pesewas`, `buyer_name`, `payment_method`, `ledger_entry_id` |

Inventory is **derived** — no stored table. Computed via the SQL query in file:specs/05_EGG_PRODUCTION.md §5.4.

### Business Rules (all from file:specs/05_EGG_PRODUCTION.md §8)

- **R1 — Eligibility:** `species='layer'` OR `species='duck' AND duck_type='layer'` only. All others → `404 EGG_TRACKING_NOT_APPLICABLE`.
- **R2 — Layer start week:** `current_week >= 19` required. Earlier → `409 BEFORE_LAY_START`.
- **R3 — Duck-layer start week:** `current_week >= 20` required.
- **R5 — One record per day:** unique index enforces; duplicate → `409 DUPLICATE_COLLECTION`.
- **R6 — Edit window:** `PATCH`/`DELETE` only within 24h of `recorded_at`.
- **R9 — Sale price:** `price_per_loose_pesewas = round(price_per_crate_pesewas / 30)`.
- **R10 — Inventory non-negative:** sale that drives `goodEggsOnHand < 0` → `409 INSUFFICIENT_EGG_INVENTORY`.
- **R11 — Withdrawal guard:** sale during active withdrawal → `409 WITHDRAWAL_ACTIVE` (even with `acknowledgeWithdrawal=true` — sale is still rejected; only discard is allowed).
- **R13 — Auto-revenue:** `POST /egg-sales` publishes `EGG_SALE_RECORDED` → Finance consumer creates `revenue_entries` row and back-patches `egg_sales.ledger_entry_id`.
- **R14 — Cost privacy:** `price_per_crate_pesewas` and `total_revenue_pesewas` stripped from list/detail when privacy active.

## Acceptance Criteria

- Layer Week 18 → `409 BEFORE_LAY_START`; Week 19 → 201
- Duck-layer Week 19 → `409 BEFORE_LAY_START`; Week 20 → 201
- Broiler batch → `404 EGG_TRACKING_NOT_APPLICABLE`
- Turkey batch → `404 EGG_TRACKING_NOT_APPLICABLE`
- Duplicate `(batch_id, collection_date)` → `409 DUPLICATE_COLLECTION`
- `PATCH` beyond 24h → `409 EDIT_WINDOW_EXPIRED`
- Sale with active withdrawal → `409 WITHDRAWAL_ACTIVE`
- Sale with `goodEggsOnHand` insufficient → `409 INSUFFICIENT_EGG_INVENTORY`
- `EGG_SALE_RECORDED` published on successful sale
- Finance consumer creates `revenue_entries` row; `egg_sales.ledger_entry_id` back-patched
- Privacy active → `pricePerCratePesewas` and `totalRevenuePesewas` absent from response
- `bun run typecheck` passes
